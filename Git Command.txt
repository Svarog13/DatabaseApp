//показує зміни, які є на даний момент до коміту
git status

//Червноий колір - це файли, які не додані до коміту
//Зелений колір - це файли, які є додані у коміт, тобто при коміту будуть комінитися і фіксуватия зміни.


git add .
//. - значить, що усі файл, які є червоні стануть зелені і будуть додані у коміт. Тобто стануть зелені.

git commit -m"Text commit"

//Робимо точку фіксації змін, до якої потім можна відкотитися назад.
//Кожен коміт має дату і час фіксації, хто зробив даний коміт(email і username).
Для того, що закинути коміт на гіт хаб робимо git push, 
для того, щоб потягнуть із гіт хаба зміни робимо git pull.

Щоб правильно працювали команди git push або git pull - потібно, щоб зміни були закомічені, або скасовані.

Для скасування змін виконується команда git stash - будуть відмінятися усі зміни, які не закомічені.
